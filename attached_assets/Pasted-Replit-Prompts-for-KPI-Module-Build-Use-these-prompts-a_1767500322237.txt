Phase B: Core KPI Calculations and API
Copy and paste this prompt:
Continue building the KPI Module. Phase A (database) is complete.

Now build Phase B: Core KPI Calculations and API Endpoints

CALCULATIONS TO IMPLEMENT:

1. Daily Labor Revenue (per tradesman):
   - Sum labor amounts from completed tasks/jobs for the day
   - Compare against daily_labor_target from user record

2. Weekly Labor Revenue (per tradesman):
   - Sum of daily labor revenues for the week
   - Compare against weekly_labor_target

3. Close Rate (per tradesman, per period):
   - quotes_accepted / quotes_sent x 100
   - Pull from existing quotes table

4. Job Profitability:
   - job_profit = labor_revenue - (hours_logged x user.hourly_cost_loaded)
   - daily_profit_rate = job_profit / actual_duration_days

5. Duration Variance:
   - actual_duration = completed_at - start_date
   - variance = actual_duration - quoted_duration

6. Schedule Capacity:
   - For each period (this week, +1, +2, +3, +4 weeks)
   - booked_days / available_days (5) x 100

API ENDPOINTS TO CREATE:

GET /api/v1/kpi/dashboard/daily
GET /api/v1/kpi/dashboard/weekly  
GET /api/v1/kpi/dashboard/monthly
GET /api/v1/kpi/dashboard/owner (requires owner role)
GET /api/v1/kpi/tradesman/{user_id}/summary
GET /api/v1/kpi/pipeline
GET /api/v1/kpi/capacity

All endpoints return standard response format with success/data/error structure.

See attached KPI Module document for full calculation formulas and response formats.

- 4+ weeks hitting $10k weekly labor
- Completed sales coaching sessions
- Demonstrated phone qualification skills
- Business Owner sign-off

Phase 2 → 3 Checklist Items:
- 8+ weeks at Phase 2
- $50k+ drainage jobs closed independently
- Shadowed 3+ bathroom sales
- Business Owner sign-off

UI COMPONENTS:
- Phase progression screen per tradesman
- Visual timeline showing phase history
- Interactive checklist (owner can check items)
- "Advance Phase" button (active only when all items complete)

API ENDPOINTS:
GET /api/v1/kpi/tradesman/{user_id}/phase
POST /api/v1/kpi/tradesman/{user_id}/phase/advance (owner only)
POST /api/v1/kpi/tradesman/{user_id}/phase/checklist/{item_id} (toggle item)

BONUS CALCULATION:

Phase 2 Tiers (labor value of closed sales per week):
- $10,000 - $12,000: $0 (base KPI)
- $12,001 - $14,000: $300
- $14,001 - $16,000: $500
- $16,001 - $18,000: $750
- $18,001+: $1,000

Phase 3 Tiers:
- $10,000 - $12,000: $0 (base KPI)
- $12,001 - $15,000: $400
- $15,001 - $18,000: $700
- $18,001 - $22,000: $1,000
- $22,001+: $1,500

API ENDPOINTS:

SCHEDULED JOB:
